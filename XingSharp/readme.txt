-------------------------------------------------------------------------------
< 개요 >
-------------------------------------------------------------------------------

이 라이브러리는 이베스트 증권 xing api 의 C# 라이브러리 입니다.

기존의 증권사에서 제공하는 activex로 된 라이브러리를 사용하여

C#에서 개발시 메모리 관리 문제로 인해 다양한 문제가 발생하여 

실제 거래 프로그램을 제작하기에 많은 어려운점이 있습니다.

이러한 문제점을 해결하기 위해 DLL을 직접 사용 하여 래핑 하였습니다.


-------------------------------------------------------------------------------
< 중요 사항 >
-------------------------------------------------------------------------------

하단의 내용에 모두 동의 하는 경우에만 사용하여 주시기 바랍니다.

- 오랜기간 충분한 모의 테스트를 거쳐 사용하시기 바랍니다.

- 이 라이브러리를 사용하여 발생하는 모든 결과는, 사용하는 당사자에게 귀속됩니다.

- 이 라이브러리를 사용함에 있어 발생하는 손해에 대하여 개발자는 어떠한 책임(금전적인 부분 포함)도 지지 않습니다. 

- 이 라이브러리는 버그가 있을 수 있습니다.

- 이 라이브러리는 업데이트를 보장하지 않으며, 사용자의 요청사항에 대해 개발/해결할 의무가 없습니다.

- 개발과 관련한 문의는 이베스트 투자증권 문의 게시판을 이용해 주세요.
